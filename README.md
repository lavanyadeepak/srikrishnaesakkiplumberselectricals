# Sri Krishna Plumbers and Electricals Website

## Overview
This repository contains a simple static webpage created for **Sri Krishna Plumbers and Electricals**, a service-based business run by **Esakki Krishna Sudalai Yadav**.  
The page highlights services such as **Electrical Wiring, Plumbing, Motor Service, Installation**, and more, along with contact information for booking.

## How to Use This Template for Your Business

1. **Fork this repository**
2. **Edit `index.html`:**
   - Replace business name and contact details
   - Update WhatsApp number in the order form
   - Modify the announcement section
3. **Update products:**
   - Edit the `products` array in the `<script>` section of `index.html`
4. **Customize colors/branding:**
   - Modify CSS variables in `styles.css`
5. **Deploy to GitHub Pages:**
   - Go to Settings → Pages → Select main branch
   - Your site will be live at `https://yourusername.github.io/yourrepo`

## Features
- 📜 **Welcome Message** – Introduction to the proprietor and experience.  
- 📢 **Announcements** – Service updates and emergency availability.  
- 🛠️ **Service Catalog** – Organized listing of services with:
  - Service name (Tamil + English)
  - Unit/Type
  - Estimated Rate
- 📝 **Booking Form** – Easy form to request services.  
- 📞 **Contact Information** – Phone number for bookings.  
- © **Footer** – Copyright notice.

- ## WhatsApp Order Integration
Service requests are automatically formatted and sent via WhatsApp with:
- Customer details (name, mobile, address)
- GPS location (if permitted)
- Itemized service list
- Total estimated cost

**To customize:** Update the WhatsApp number in the `script` section of `index.html`.
